Bạn đang làm việc trong project Kaggle “ROGII - Wellbore Geology Prediction”.

Mục tiêu project:
- Xây dựng pipeline dự đoán TVT bị khuyết trong test bằng Autoregressive LightGBM.
- Dự báo theo kiểu Recursive Multi-step Forecasting: với mỗi row thiếu `TVT_input`, model dự đoán `TVT`, rồi ghi ngay prediction đó vào history để tính lag/rolling cho các row sau.
- Tránh leakage: mặc định loại 3 test well IDs khỏi training: `000d7d20`, `00bbac68`, `00e12e8b`.

Cấu trúc hiện tại:
- `data/`: chứa data thật từ Kaggle.
  - `data/train/`: 773 wells, mỗi well có `<well_id>__horizontal_well.csv`, `<well_id>__typewell.csv`, và PNG.
  - `data/test/`: 3 wells, mỗi well có horizontal/typewell CSV.
  - `data/sample_submission.csv`: schema `id,tvt`, id dạng `<well_id>_<row_index>`.
- `src/dataset.py`: đọc well pairs, parse `well_id`, giữ `row_index`, đọc sample submission.
- `src/features.py`: tạo feature autoregressive.
- `src/model.py`: LightGBM factory, save/load artifacts.
- `src/train.py`: CLI train holdout/groupkfold.
- `src/predict.py`: CLI recursive inference và tạo submission.
- `notebook/`: notebook skeleton cho EDA, Colab train, local inference, error analysis.
- `tests/`: unit tests cho feature leakage và recursive update.

Ràng buộc dữ liệu quan trọng:
- Train horizontal có cột:
  `MD,X,Y,Z,ANCC,ASTNU,ASTNL,EGFDU,EGFDL,BUDA,TVT,GR,TVT_input`
- Test horizontal chỉ có:
  `MD,X,Y,Z,GR,TVT_input`
- Vì vậy model submit KHÔNG được dùng `ANCC, ASTNU, ASTNL, EGFDU, EGFDL, BUDA`.
- Typewell test chỉ có `TVT,GR`; không có `X,Y,Z`, nên không thể tính Euclidean distance 3D tới typewell. Thay bằng feature alignment theo TVT/GR:
  `typewell_gr_at_nearest_tvt`, `gr_minus_typewell_gr`, `nearest_typewell_tvt_gap`, rolling stats của typewell GR quanh TVT gần nhất.

Feature engineering hiện có:
- Base inputs: `MD`, `X`, `Y`, `Z`, `GR`.
- History columns: `TVT`, `X`, `Y`, `Z`, `GR`.
- Lags: `[1,2,3,5,10,20]`.
- Rolling mean/std windows: `[5,10,20]`.
- Spatial derivatives: `dX`, `dY`, `dZ`, `dMD`, `xy_step_distance`, `step_distance`, `path_distance`, `dZ_dMD`.
- Tất cả TVT lag/rolling phải dùng quá khứ trước row hiện tại, tuyệt đối không dùng target hiện tại.
